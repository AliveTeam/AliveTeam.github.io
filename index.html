R.E.L.I.V.E.

An Open-Source Abe's Oddysee/Exoddus Engine Reimplementation

What? Why?

R.E.L.I.V.E. is a fan effort to create a new engine, capable of replacing Digital Dialect's aging PC port for Oddworld's first two cult-classic games.

More than twenty years after their release, both games struggle to run on modern platforms, usually requiring a third-party application like DxWnd just to run them properly.

R.E.L.I.V.E. intends to become a complete replacement of both the original AbeWin.exe and any third-party tools, by providing both convenience and compatibility features like:

  • Running the game in native resolutions,

  • built-in support for switching between fullscreen and windowed mode,

  • Native support for both MacOS and Linux,

  • The re-write uses the modern and cross-platform SDL2 Graphics Library, instead of the long deprecated DirectDraw API.

  • A fully open-source codebase, which you can freely use for your own projects or extend as you see fit.

Here be Scrabs!

While the part of the engine responsible for Exoddus is playable from start to finish already, the experience is currently not bug-free. You can see the list of OG and reimplementation-caused bugs here. Also native gamepad support currently only works on Windows.

The Future

After the engine is fully reimplemented, the focus will shift on new features that which were never part of the original.

  • More platforms (sneak peek).
  • Subtitles for cutscenes.
  • Fixing the original games' bugs.
  • Modding support.
  • Level editor.
  • Higher-resolution textures.
  • Feature requests by the community.
